<h1>📄 Personal Portfolio</h1>

<p>This is a full-stack personal portfolio project that showcases skills, projects, and more. It includes a powerful admin panel for managing portfolio content like skills and projects. The project is built with modern web technologies, providing a seamless user experience with dynamic animations and responsive design.</p>

<h2>🔗 Live Demo</h2>
<p>Check out the live version of the project here: <a href="https://personal-portfolio-delta-flax-13.vercel.app" target="_blank">Live Demo</a></p>

<h2>🚀 Features</h2>
<ul>
  <li><strong>🎨 Portfolio Showcase:</strong> Display personal projects, skills, and accomplishments.</li>
  <li><strong>🔐 Admin Panel:</strong> Admins can manage skills and projects dynamically through a secure admin panel.</li>
  <li><strong>✨ Dynamic Animations:</strong> Smooth transitions and animations using GSAP.</li>
  <li><strong>📱 Responsive Design:</strong> Mobile-friendly UI built with Tailwind CSS and Shadcn UI.</li>
  <li><strong>⚡ Fast Data Fetching:</strong> Efficient data fetching and caching using React Query.</li>
  <li><strong>🌗 Dark Mode Support:</strong> Tailored user experience with light and dark themes.</li>
  <li><strong>🔒 Secure Authentication:</strong> Authentication for the admin panel to manage skills and projects.</li>
</ul>

<h2>🛠️ Tech Stack</h2>

<h3>Frontend (written in TypeScript):</h3>
<ul>
  <li>⚛️ <strong>React.js:</strong> Component-based architecture for building the UI.</li>
  <li>🎯 <strong>React Query:</strong> Handles server-side data fetching and caching.</li>
  <li>💨 <strong>Tailwind CSS:</strong> Utility-first CSS framework for responsive design.</li>
  <li>🔧 <strong>Shadcn UI & Magic UI:</strong> For modern and customizable UI components.</li>
  <li>🌀 <strong>GSAP:</strong> Animation library for smooth transitions and effects.</li>
</ul>

<h3>Backend (written in TypeScript):</h3>
<ul>
  <li>🟢 <strong>Node.js & Express.js:</strong> Backend server and API development.</li>
  <li>🍃 <strong>MongoDB:</strong> Database for storing portfolio data like skills and projects.</li>
</ul>

<h2>🔑 Features for Admin:</h2>
<ul>
  <li>Add, update, and delete <strong>skills</strong>.</li>
  <li>Add, update, and delete <strong>projects</strong>.</li>
  <li>Secured with authentication.</li>
</ul>

<h2>📦 Installation</h2>

<h3>Prerequisites:</h3>

  <li>Node.js</li>
</ul>
